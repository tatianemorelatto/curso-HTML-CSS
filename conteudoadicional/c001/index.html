Downloads de arquivos em sites

Para downloads, as âncoras recebem também o atributo download com o valor configurado para o nome do arquivo a ser baixado, e o atributo type para indicar ao navegador que tipo de arquivo está sendo baixado.

Exemplo:

Baixe aqui o PDF do meu livro

Aqui vão alguns media types bem usados no nosso dia a dia

  • application/zip (para pastas ou arquivos zips)
  • video/mp4
  • audio/mpeg
  • text/html
  • text/css
  • text/javascript
  • video/H264
  • video/JPEG
  • audio/aac
  • font/ttf
  • image/jpeg
  • image/png
