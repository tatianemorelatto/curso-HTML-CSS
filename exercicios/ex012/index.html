Compatibilidade em NAVEGADORES

Atualmente existem padrões utilizados e suportados em navegadores, são eles:

  • MP4 (.MP4 e .M4V)
  • webm (sofrivel)
  • ogv (ogg)

Os formatos MP4/M4V e WEBM. são os que possuem maior compatibilidade com os navegadores atualmente.

Navegadores e Arquivos compatíveis

  • Edge
    • (.mp4 / .m4v)
  • Safari (Apple)
    • (.mp4 / .m4v)
  • Chrome
    • (.mp4 / .m4v / .webm / .ogv)
  • Mozila Firefox
    • (.webm / .ogv)
  • Opera
    • (.webm / .ogv)

Inserindo vídeos hospedados localmente

Este vídeo está hospedado no meu próprio servidor.

Como vimos acima, cada navegador é compatível com um arquivo, por isso, é importante adicionar todos os arquivos para o mesmo vídeo.

Isso porque o usuário pode acessar seu site do Chrome, do Safari, Firefox, ou qualquer outro, e não conseguir reproduzir o vídeo, pois o formato que você inseriu, não é compativel.

Lembrando que fazendo dessa forma estamos hospedando localmente, podem e vão consumir muito tráfego de dados.
